# 位，字节及排列组合
计算机底层都是二进制，也就是由零或者一构成的。

位bit，要么放零，要么放一。

一个字节=8位，也就是1Byte=8bit。

n个位也就有2的n次方个排列组合。

1KB=1024B，1M=1024KB，1G=1024M, 1T=1024G。

如果要在运行程序后，在界面展示我们往往会用到printf（）函数，以下是一个示例
